# NewsStockAnalysis
新聞與股票分析系統 - 畢業專題

記得在自己的分支(branch)上，進行作業，完成上傳分支，後面待測試無誤後一並併入main
作業前記得先fetch再pull(下載)最新檔案下來(避免版本衝突)
commit 記得打

有在每個資料夾中放入readme.md ，這裡要記錄，如果今天我想執行你們的檔案要怎麼做 ↓例
